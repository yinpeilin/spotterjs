# @spotter/core

Primary TypeScript API for Spotter desktop automation (screen, mouse, keyboard, windows, accessibility).

Requires `@spotter-rs/node` native addon (prebuilt for Windows x64 and Linux x64-gnu).

```bash
npm install @spotter/core
```
